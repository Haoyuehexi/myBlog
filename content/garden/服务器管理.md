## 服务器创建新用户
用已有管理员账户登录服务器后，执行：
```bash
sudo adduser new_username
```
可以看见```/home```文件夹下出现新的用户名


## 给新用户权限
如果你希望该用户也能使用```sudo```：
```bash
sudo usermod -aG sudo new_username
```
